# Snake Game

This is a desktop version of the famous Snake Game created in Python using the Tkinter library.

## Run Locally

Clone the project:

```bash
git clone https://github.com/johnnyweak1/Python-SnakeGame
```

Go to the project directory:

```bash
cd Python-SnakeGame
```

Run script:


```bash
python3 main.py
```
